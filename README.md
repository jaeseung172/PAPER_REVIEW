# 논문 리뷰

2017년도 안에 읽었던 논문들을 짤막한 슬라이드로 리뷰하여 올려놓았습니다, 관련 연구하실 때 참고 부탁드립니다. 감사합니다.

아직 생물학을 전공하지 / 통계, 머신러닝, 딥러닝을 전공하지 않은 고등학생인지라 아리까리한 곳은 몇번씩 읽고 관련 논문을 더 찾아서 읽어가며 손보는 편이지만 그래도 문제가 발생할 수도 있습니다. 양해 부탁드립니다. 감사합니다.
